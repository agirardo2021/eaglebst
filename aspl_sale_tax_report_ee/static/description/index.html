Odoo 15 Sale Tax Report (Enterprise)

Note: This module is tested and working in Enterprise Edition. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Generate Sale Tax Report in PDF.
  • You can view report using group by tax.

Available for Community

How to use

  • Go to the Invoicing -> Custom Report -> Sale Tax Report.

  • Select Date Range to get report accordingly.
  • Select Taxes which have sales tax type.
  • Select States to get invoices for that state.

  • Sale Tax Report
